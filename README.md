# Simple Calculator

A modern, interactive calculator built with HTML, CSS, and JavaScript. Includes optional backend implementations with Node.js/Express and Python/Flask.

## Features

- ✨ Clean and modern UI
- 🧮 Basic arithmetic operations (add, subtract, multiply, divide)
- 📱 Responsive design
- ⌨️ Keyboard support
- 📊 Calculation history
- 🔄 Clear and reset functions

## Quick Start

### Frontend Only

1. Navigate to the `frontend` folder
2. Open `index.html` in your browser
3. Start calculating!

### With Node.js Backend

1. Navigate to `backend-nodejs`
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` in your browser

### With Python Backend

1. Navigate to `backend-python`
2. Run `pip install -r requirements.txt`
3. Run `python app.py`
4. Open `http://localhost:5000` in your browser
